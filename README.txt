1. https://github.com 가입 (무료)
2. 새 Repository 생성 (예: contec-pedometer)
3. index.html 과 logo.png 업로드
4. Settings → Pages → Branch: main, folder: / (root) → Save
5. 잠시 후 https://[계정명].github.io/contec-pedometer 로 접속 가능

※ 직원들에게 URL 공유 → 암호 알려주면 바로 사용 가능
